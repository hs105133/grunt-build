# Grunt build with Angular JS

- clean build directory
- concatenation js & css
- copy extra stuff like fonts, images, favicon.ico to build directory
- css minification
- js minification
- image minification
- js code linting 
- watching js, css, html, less file changes
- code versioning
- Angular JS code annotation

## Run Application

```
node server.js
```

## Build Step

```
grunt
```
